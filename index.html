Este sitio presenta una prueba base para la visualización de modelos 3D en navegador y su despliegue en realidad aumentada desde dispositivos móviles compatibles. El objetivo es servir como referencia técnica inicial para futuros proyectos académicos y de experimentación en entornos web inmersivos.

Archivo actual: smartwatch.glb

Cómo usar este demo

  • Abre esta página desde un celular compatible.
  • Espera a que cargue el modelo 3D.
  • Explora el objeto directamente en el visor.
  • Activa el modo de realidad aumentada con el botón del visor.
  • Apunta hacia una superficie estable para posicionar el modelo en el espacio.

Propósito académico

Este demo funciona como una base explicativa para comprender cómo un modelo digital puede ser publicado en web, visualizado en tres dimensiones y proyectado en realidad aumentada desde el celular. Su intención es apoyar el desarrollo de ejercicios iniciales, pruebas de concepto y propuestas académicas de carácter exploratorio.
